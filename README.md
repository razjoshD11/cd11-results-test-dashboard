# CD-11 Public Results Dashboard — TEST harness

A test deployment of the public CD-11 results map, driven by the **March 5, 2024
CD-11 top-two primary** (8 candidates) as a parallel to the June 2026 primary.

Use the switcher bar at the top to step through the simulated election-night
ballot drops:

- **Drop 1** (~8:45pm, first VBM dump) — citywide totals only, no precinct map.
- **Drop 2** (~11pm, + election day) — citywide totals only.
- **Final** (certified) — full per-precinct breakdown over all SF precincts.

The map shows the top-3 vote-getters per precinct, supervisor district, and
citywide. Hover a precinct for the leader + share; click for the breakdown.

Real certified citywide totals (CA SOS / Ballotpedia); per-precinct counts are
synthetically distributed and reconcile exactly to each candidate's real total.
This is a TEST surface — not official results.
